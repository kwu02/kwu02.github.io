Course Title
The Complete Web Developer in 2021: Zero to Mastery taught by Andrei Neagoie


Course Description
A comprehensive course shows you how to code and become a web developer with HTML, CSS, JavaScript, React, Node.js, Machine Learning and more.



Who should take this course?
It’s not only limited to web developer, people who would like to apply jobs like software developer, front end developer, JavaScript developer, and full stack should take advantage of this course. In addition to HTML 5 and CSS which are learning in CNIT 132, this course also covers topics like JavaScript and DOM manipulation that we will learn in CNIT 133 and other advance classes. This course also shows us how to use best practices to write clean, performant, and bug free code.


What we will learn from this course?

Front-end development

  • HTML 5
  • CSS
  • JavaScript
  • React.js and React hooks
  • JSON
  • and more

Back-end development

  • API
  • Node.js and Express.js
  • SQL
  • and more

My progress
Due to my work and class schedule, I have only completed the HTML, CSS, CSS grid, CSS layout, and JavaScript sections so far. However, the materials did give me some ideas to complete the homework assignments in CNIT 132 and CNIT 133. I am planning to complete other sections such as DOM manipulation, NPM, NPM scripts, reacts, and JSPON/AJAX during the summer break and prepare myself for the classes I am taking next.


In conclusion
This course will not replace these great classes offered at CCSF, but it’s a good tutorial that can help us the materials. It’s a self-paced learning, and I can focus on things that I find challenging.
